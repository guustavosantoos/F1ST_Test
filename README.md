# Projeto de Automação de Testes - E-commerce & API

## Instalação dos Requisitos:

```
1. git clone https://github.com/guustavosantoos/F1ST_Test.git 
 
2. npm init -y 

3. npm install cypress@13.7.0 --save-dev  
```


## Execução dos Testes

**Modo interativo:**

```
npx cypress open
```


**Modo headless:**

```
npx cypress run
```
